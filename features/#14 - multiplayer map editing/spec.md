## Purpose

When more than one player (including DM and Owner) is on the map at the same time, their edits to the map and grid
overlay should be immediately seen by all other players on the map in real-time. This includes:

- Map-level variables, though these currently don't exist.
- Grid overlay element (cell) variables. Right now, this only includes the cell name, but more variables will be added
  in the future. This solution should automatically deal with any additional parameters added to cells, if possible.
- Player / DM / Owner online status - show if a player is online or not.
- Mouse position - show the mouse position of other players.
- Selection - show the current selection of other players. This includes if a player selects a cell and should also add
  a border effect on the field a player is editing in the fly-out panel.
